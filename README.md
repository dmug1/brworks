# BrWorks

## This project aims to help the Brazilian community in Australia.


## Built With
* [Vue.Js]
* [SCSS]


## Project setup
```
cd app-brworks
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
